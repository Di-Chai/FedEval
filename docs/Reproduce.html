4. Reproduce the results in the paper¶

4.1. Reproducing scripts¶

We provide the following scripts for reproducing the results in our paper

Experiments Reproducing Scripts
Local and central training Local_Central.py
Grid search of B, C, and E in FedAvg Dataset_Tune_BCE.py
Tuning the optimizers and learning rates Dataset_Tune_Op_LR.py
Accuracy, Communication, Time consumption Evaluation Dataset_Eval.py
Privacy Evaluation DLG_Attack_MultiThread.py
DLG_Attack_Metric.py
FC_Attack_MultiThread.py
FC_Attack_Metric.py
Robustness Evaluation Dataset_Non_IID.py

where the Dataset can be MNIST, FEMNIST, and CelebA, e.g., MNIST_Tune_BCE.py.

4.2. Running the scripts¶

Move to the PaperScripts dir

cd PaperScripts

1 Local training and central training

docker run --rm -v $(pwd):$(pwd) -w $(pwd) fleval:v1 python Local_Central.py

2 Grid search of B, C, and E in FedAvg

# 2(1) MNIST Dataset, output file: MNIST_Tune_BCE_trials.txt
python MNIST_Tune_BCE.py
# 2(2) FEMNIST Dataset, output file: FEMNIST_Tune_BCE_trials.txt
python FEMNIST_Tune_BCE.py
# 2(3) CelebA Dataset, output file: CelebA_Tune_BCE_trials.txt
python CelebA_Tune_BCE.py

3 Tuning the optimizers and learning rates

# 3(1) MNIST Dataset, output file: MNIST_Tune_Op_LR_trials.txt
python MNIST_Tune_Op_LR.py
# 3(2) FEMNIST Dataset, output file: FEMNIST_Tune_Op_LR_trials.txt
python FEMNIST_Tune_Op_LR.py
# 3(3) CelebA Dataset, output file: CelebA_Tune_Op_LR_trials.txt
python CelebA_Tune_Op_LR.py

4 Accuracy, Communication, and Time consumption evaluation.

# 4(1) MNIST Dataset, output file: MNIST_Eval_trials.txt
python MNIST_Eval.py
# 4(2) FEMNIST Dataset, output file: FEMNIST_Eval_trials.txt
python FEMNIST_Eval.py
# 4(3) CelebA Dataset, output file: CelebA_Eval_trials.txt
python CelebA_Eval.py

5 Privacy experiments, you can change “–cpus” value to increase or decrease the cpu resources used in the experiments.

# 5(1.1) Start the DLG attack using multi-threads
docker run --rm --cpus 8 -v $(pwd):$(pwd) -w $(pwd) fleval:v1 python DLG_Attack_MultiThread.py
# 5(1.2) Calculate the label accuracy and L2 distance
docker run --rm -v $(pwd):$(pwd) -w $(pwd) fleval:v1 python -W ignore DLG_Attack_Metric.py
# 5(2.1) Start the FC attack using multi-threads
docker run --rm --cpus 8 -v $(pwd):$(pwd) -w $(pwd) fleval:v1 python FC_Attack_MultiThread.py
# 5(2.2) Calculate the label accuracy and L2 distance
docker run --rm -v $(pwd):$(pwd) -w $(pwd) fleval:v1 python -W ignore FC_Attack_Metric.py

6 Robustness evaluation

# 6(1) MNIST Dataset, output file: MNIST_Non_IID_trials.txt
python MNIST_Non_IID.py
# 6(2) FEMNIST Dataset, output file: FEMNIST_Non_IID_trials.txt
python FEMNIST_Non_IID.py
# 6(3) CelebA Dataset, output file: CelebA_Non_IID_trials.txt
python CelebA_Non_IID.py

The outputs of experiments 1-4 and 6 are in forms of txt files. Here is the example outputs, and the results can also be found on the dashboard page. The outputs of experiments 5 are images, and the attack label accuracy and L2 distance will be printed when running the metric scripts (e.g., DLG_Attack_Metric.py).
