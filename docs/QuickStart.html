2. Quick Start¶

Move to the QuickStart dir

cd QuickStart

2.1. Generate the data and docker-compose file¶

Generate the data for all the clients

docker run --rm -v $(pwd):$(pwd) -w $(pwd) fleval:v1 python 4_distribute_data.py

Generate the docker-compose file

docker run --rm -v $(pwd):$(pwd) -w $(pwd) fleval:v1 python 5_generate_docker_compose.py

2.2. Start the Experiment¶

docker-compose up -d

2.3. View the results¶

Go to http://127.0.0.1:8200/dashboard to see the results. The default port is 8200, and you can change the default settings in the Configs.

Here’s an example of the dashboard.

2.4. Stop the Experiment¶

Stop and remove the containers using docker-compose

docker-compose stop
docker-compose rm

Force stop and remove all the containers

docker stop $(sudo docker ps --filter ancestor=fleval:v1 -aq)
docker rm $(sudo docker ps --filter ancestor=fleval:v1 -aq)
